# Business Author Style Profile

Style target: $ARGUMENTS

## Purpose

Create `AUTHOR_STYLE_PROFILE.md` from author samples and target journal norms, then apply it during `business-paper-writing`.

## Inputs

Use available materials:

- prior papers, dissertation chapters, working papers, or class notes by the author
- target journal exemplars
- advisor or coauthor examples
- explicit style preferences from the user
- `BUSINESS_PAPER_PLAN.md` and `CLAIMS_FROM_EVIDENCE.md` when drafting is next

Read `../shared-references/business-style-calibration.md` for dimensions and template.

## Workflow

### Step 1: Separate Norms

Identify:

- author habits
- target journal conventions
- field conventions
- unsuitable quirks to avoid

Journal and discipline norms override personal style.

### Step 2: Profile Style Dimensions

Capture:

- sentence and paragraph length
- hedging level
- citation integration
- theory and mechanism phrasing
- table-result narration
- economic magnitude language
- preferred transitions

### Step 3: Convert To Drafting Rules

Write rules that `business-paper-writing` can apply without imitating errors or overstating evidence.

### Step 4: Apply With Claim Discipline

When drafting, keep `CLAIMS_FROM_EVIDENCE.md`, `BUSINESS_NUMBER_AUDIT.md`, and `SOURCE_CLAIM_AUDIT.md` above style preferences.

## Output

Create or update `AUTHOR_STYLE_PROFILE.md`:

```markdown
# Author Style Profile

## Source Samples
## Discipline Baseline
## Style Dimensions
## Reusable Sentence Moves
## Result-Narration Preferences
## Citation Style Preferences
## Guardrails
```

## Rules

- For local tasks, complete only the requested stage and mark downstream gaps as next-stage inputs.
- Do not mimic unclear, inflated, or reviewer-hostile habits.
- Do not change evidence strength to match style.
- Preserve concise business-journal prose.
