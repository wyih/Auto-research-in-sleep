# Business Style Calibration

Use this protocol when a user provides prior writing samples, target journal articles, advisor examples, or a house style.

## Core Rule

Journal and discipline norms override personal style. Preserve the author's durable voice only within the range accepted by accounting, finance, management, economics, or business-school journals.

## Profile Dimensions

Capture:

- sentence length and complexity
- paragraph length and topic-sentence pattern
- hedging strength
- transition style
- citation integration style
- reporting verbs
- theory and mechanism language
- table-result narration style
- use of economic magnitude
- tolerance for first-person or passive voice
